## Instructions d'import dans Microsoft Forms

1. Aller sur **forms.microsoft.com** → « Nouveau formulaire »
2. Titre : **Demande d'offre — Responsabilité Civile Professionnelle**
3. Description : *Ce formulaire permet à DAP d'analyser les risques de votre activité professionnelle et de vous proposer une couverture RC Pro adaptée au marché belge. Toutes les informations sont traitées de façon confidentielle.*
4. Créer chaque question dans l'ordre indiqué ci-dessous
5. Pour les **sections conditionnelles** : utiliser l'option « Ajouter une branche » dans MS Forms
6. Activer « Obligatoire » (toggle) pour chaque question marquée ✅

## Points d'attention spécifiques RC Pro (marché belge)

> À communiquer au client avant/après remplissage :

- **Base claims made** : La RC Pro couvre les réclamations **présentées** pendant la période de validité du contrat, pas nécessairement quand l'acte générateur s'est produit. Anticiper la date de prise d'effet.
- **Extension rétroactive** : Négocier une date rétroactive (backdating) pour couvrir des actes passés si l'entreprise n'était pas assurée avant.
- **USA/Canada** : Toute activité (même marginale) avec des clients américains ou canadiens doit être déclarée — surprime systématique et certains assureurs refusent ce risque.
- **Garanties imposées** : Les membres de l'Ordre des Architectes, du Barreau et de l'ITAA ont des minimums légaux de couverture — vérifier avant de proposer un capital.
