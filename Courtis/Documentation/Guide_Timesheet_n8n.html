Guide d'installation — Timesheet Automatique

Outil de génération automatique de timesheets via Microsoft 365 (Outlook, Teams, Calendrier)

Version 1.0 — Mars 2026 — DAP

À quoi sert cet outil ?

Vous tapez le nom d'un client ou d'un dossier. L'outil fouille automatiquement vos mails, messages Teams et réunions sur la période choisie. Vous sélectionnez ce que vous voulez facturer, saisissez vos heures et votre tarif. Un fichier Excel est généré automatiquement et sauvegardé dans votre OneDrive.

Saisie du pivot → Recherche M365 → Revue des interactions → Heures + tarif → Export Excel OneDrive

Étape 1 — Créer un compte n8n gratuit

1
Aller sur n8n.cloud

Ouvrez votre navigateur et accédez à https://n8n.io, cliquez sur Get started free et créez un compte avec votre adresse email professionnelle.

2
Accéder à votre espace n8n

Une fois connecté, vous arrivez sur votre tableau de bord. Le plan gratuit est suffisant pour cet outil.

Étape 2 — Importer le workflow

1
Récupérer le fichier workflow

Demandez le fichier Timesheet_Automatique_Microsoft365.json à votre collègue.

2
Importer dans n8n

Dans n8n, cliquez sur + New workflow, puis sur les trois points (...) en haut à droite → Import from file. Sélectionnez le fichier JSON reçu.

3
Sauvegarder

Cliquez sur Save en haut à droite.

Étape 3 — Connecter votre compte Microsoft 365

Créer la credential dans n8n

Dans n8n : Settings → Credentials → New, cherchez Microsoft OAuth2 API et cliquez dessus.

3
Autoriser la connexion

Cliquez sur Connect. Une fenêtre Microsoft s'ouvre — connectez-vous avec votre compte professionnel DAP et acceptez les permissions.

4
Assigner la credential aux 4 nœuds HTTP

Dans le workflow, cliquez sur chacun de ces nœuds et sélectionnez la credential que vous venez de créer :

  • Recherche Emails
  • Recherche Teams
  • Recherche Réunions
  • Upload OneDrive

Étape 4 — Utiliser l'outil au quotidien

1
Activer le workflow

Cliquez sur Publish en haut à droite. Le workflow est maintenant actif en permanence avec un lien fixe.

2
Ouvrir le formulaire

Cliquez sur le nœud Saisie Pivot pour récupérer l'URL du formulaire. Mettez-la en favori dans votre navigateur.

3
Remplir le formulaire de recherche
ChampExemple
Pivot de rechercheJean Dupont / POL-2024-001 / RC Pro
Date de début01/01/2026
Date de fin28/02/2026
Fenêtre Teams ± heures72 (laisser vide = 72h par défaut)
4
Sélectionner les interactions à facturer

Une liste chronologique s'affiche avec tous les emails, messages Teams et réunions trouvés. Cochez uniquement ceux liés au dossier client.

5
Saisir la facturation

Renseignez le nombre d'heures facturées, le tarif horaire (€/h) et une description optionnelle.

6
Récupérer le fichier Excel

Le fichier est automatiquement sauvegardé dans votre OneDrive → dossier Timesheets/. Il est prêt à être envoyé au client ou importé en comptabilité.

✅ Timesheet généré avec succès !
