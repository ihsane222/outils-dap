DAP Solidarity — Extension Chrome

Comment ça marche ?
Une fois l'URL configurée, sélectionnez le nom d'un preneur dans Brio puis cliquez sur l'icône DAP dans la barre Chrome — ou faites clic droit → Rechercher dans DAP Solidarity.

Entrez l'URL complète de l'outil DAP Solidarity déployé sur Netlify ou Azure.
Exemple : https://dap-solidarity.netlify.app
