⚡ Raccourci Brio — Installation

L'extension Chrome DAP Solidarity s'installe une seule fois sur votre navigateur. Elle vous permet de rechercher un preneur directement depuis Brio en un clic — sans changer d'onglet manuellement.

Comment utiliser l'extension
🤝
Option 1 : Sélectionnez le nom du preneur dans Brio → cliquez sur l'icône DAP dans Chrome → l'outil s'ouvre avec la recherche pré-remplie.
Option 2 : Clic droit sur le nom → "Rechercher dans DAP Solidarity".
L'extension lit automatiquement le texte sélectionné — aucune copier-coller nécessaire.
